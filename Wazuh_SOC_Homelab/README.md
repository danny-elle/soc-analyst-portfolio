#  Wazuh SOC Homelab

## Overview
My homelab simulates a Security Operations Center (SOC) enviroment using Wazuh SIEM for log analysis, alerting and incident detection.

## Goals
- Build hands on SOC experience
- Practice alert triage and incident response
- Analyze logs and detect threats

## Enviroment
- Wazuh server (Ubuntu)
- Endpoint agents (Windows 11/Kali Linux)
- Virtual Machines (VirtualBox)

## Labs
- Configurations
- Agent deployment
- Brute Force detection
